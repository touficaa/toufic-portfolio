==========================================================
 HOW TO PREVIEW THE WEBSITE
==========================================================

Simply double-click index.html OR right-click and Open With Chrome.
The site will load with placeholders where photos go.

==========================================================
 HOW TO EXPORT PDF 1
==========================================================

1. Open pdf-sources/PDF1_Athletic_Achievements.html in Chrome
2. Press CTRL + P
3. Settings:
   - Destination: Save as PDF
   - Layout: Portrait
   - Paper size: A4
   - Margins: None
   - Scale: Default
   - Background graphics: ON (very important)
4. Save the PDF to: assets/pdfs/PDF1_Athletic_Achievements.pdf

That file will then be linked from the Portfolio section of the website.
